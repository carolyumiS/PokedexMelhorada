# 📘 Pokédex em React

Aplicação simples desenvolvida em React que consome a PokéAPI para buscar e exibir informações de Pokémons.

## 🚀 Funcionalidades

* 🔍 Busca de Pokémon por nome
* 📋 Exibição de dados: nome, imagem, altura, peso e tipos
* ⭐ Sistema de favoritos
* 🔁 Lista dinâmica (Pokémons mais recentes aparecem no topo)
* 💾 Persistência de favoritos com localStorage

## 🛠️ Tecnologias

* React
* TypeScript
* Vite
* CSS

## ▶️ Como rodar o projeto

```bash
npm install
npm run dev
```

Acesse: http://localhost:5173
